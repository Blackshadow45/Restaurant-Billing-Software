🍽️ Restaurant Billing Software (Demo Version)

📌 Overview

This is a **demo version** of a restaurant billing system built to showcase core functionalities like billing, menu management, and order processing.

It demonstrates full-stack development using React, Spring Boot, and MongoDB.

------------------------------------------------------------------------------------------------------------------------------

## 🚀 Tech Stack

* **Frontend:** React.js
* **Backend:** Spring Boot (Java)
* **Database:** MongoDB Atlas
* **Deployment:** Netlify (Frontend), Render (Backend)

---------------------------------------------------------------------------------------------------------------------------------------

🔥 Features (Demo Version)

This demo includes:

* 🔐 User Registration & Login
* 📊 Dashboard
* 📋 Menu Management
* 🧾 Bill Generation
* 🖨️ KOT (Kitchen Order Ticket) Print
