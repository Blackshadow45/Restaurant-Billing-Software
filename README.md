🍽️ Restaurant Billing Software (Demo Version)

📌 Overview

This is a **demo version** of a restaurant billing system built to showcase core functionalities like billing, menu management, and order processing.

It demonstrates full-stack development using React, Spring Boot, and MongoDB.

--------------------------------------------------------------

🚀 Tech Stack

* **Frontend:** React.js
* **Backend:** Spring Boot (Java)
* **Database:** MongoDB Atlas
* **Deployment:** Netlify (Frontend), Render (Backend)

--------------------------------------------------------------------

## 🔥 Features (Demo Version)

This demo includes:

* 🔐 User Registration & Login
* 📊 Dashboard
* 📋 Menu Management
* 🧾 Bill Generation
* 🖨️ KOT (Kitchen Order Ticket) Print

-----------------------------------------------------------------------------------

## ⚠️ Limitations (Demo Scope)

This is a **limited showcase version**, so some advanced features are not included:

* Table Merge / Split
* Section-wise Pricing
* Order Shifting
* Kitchen Display System (KDS)
* Advanced Reports

---------------------------------------------------------------------------------------

💼 Production Version

The **full production version** of this software includes:

* Complete table management
* Real-time KDS (Kitchen Display System)
* Order lifecycle handling
* Table merging & shifting
* Section-based pricing
* Advanced reports

📌 This full version has been **developed and delivered to a real client in India**.

---------------------------------------------------------------------------------------------

🧪 How to Use

1. Open the application
2. Register a new account
3. Login
4. Access dashboard
5. Manage menu and create bills
